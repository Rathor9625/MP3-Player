# 🎵 AURA MUSIC PLAYER

A stylish and feature-rich **MP3 Music Player** built using  **Python, Tkinter, and Pygame** .

This player offers a modern UI with smooth controls, playlist management, and real-time progress tracking.

---

## 🚀 Features

* 🎧 Play, Pause, Resume music
* ⏭ Next / Previous track navigation
* ⏩ Seek forward & backward (±10 seconds)
* 📂 Add single songs or entire folders
* 📜 Playlist management
* 📊 Real-time progress bar
* 🔊 Volume control with percentage display
* 🎨 Modern dark UI with neon accents
* 🔁 Auto-play next song

---

## 🛠️ Technologies Used

* **Python**
* **Tkinter** (GUI)
* **Pygame** (Audio handling)
* **OS & Time modules**

## 🖥️ Usage

* Use **Library Menu** to:
* Add Song 🎵
* Add Folder 📂
* Clear Playlist ❌
* Controls:
* ▶ Play / Pause
* ⏮ Previous
* ⏭ Next
* ⏪ / ⏩ Seek 10 seconds
* Click on progress bar to jump
* Adjust volume using slider
